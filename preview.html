🚀 GitLab 周报生成器

基于 Rollup 构建的现代化 Tampermonkey 脚本

📋 开发预览说明

这是一个用于开发和测试的预览页面。在实际使用中，此脚本将作为 Tampermonkey 用户脚本运行在目标网站上。

🔧 开发工具

可用的 npm 脚本命令：

npm run build # 生产环境构建
npm run build:dev # 开发环境构建
npm run watch # 监听模式构建
npm run dev # 开发模式（带服务器）

📦 Rollup 配置特性

  • ✅ ES6 模块支持
  • ✅ CommonJS 模块转换
  • ✅ CSS 处理和注入
  • ✅ 生产环境代码压缩
  • ✅ 开发服务器支持
  • ✅ 自动添加 Tampermonkey 头部
  • ✅ 热重载开发模式

🎯 使用方法

  1. 运行 npm run dev 启动开发服务器
  2. 访问 http://localhost:3000/dist/gitlab-weekly-report.user.js 获取脚本
  3. 在 Tampermonkey 中安装该脚本
  4. 访问匹配的网站测试功能
